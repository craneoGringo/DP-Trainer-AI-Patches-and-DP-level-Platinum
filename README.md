# DP-Trainer-AI-Patches-and-DP-level-Platinum
A repository with the original BPS patches for the following ROM hacks:

Pokemon Diamond Version Trainer AI Patch

Pokemon Diamond Version Trainer AI Patch Plus

Pokemon Pearl Version Trainer AI Patch

Pokemon Pearl Version Trainer AI Patch Plus

Pokemon DP-level Platinum

Pokemon DP-level Platinum Plus

These patches were originally made to fix a bug found in the original Diamond and Pearl games, but got expanded to making slightly more challenging versions of Diamond, Pearl and Platinum based on ideas I had since a long time ago without changing much about the games themselves.

To download the patches, download the repository as a .zip file and extract.
