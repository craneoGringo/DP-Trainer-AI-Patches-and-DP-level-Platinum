# DP-Trainer-AI-Patches-and-DP-level-Platinum
A repository with the original BPS patches for the following ROM hacks:

Pokemon Diamond Version Trainer AI Patch
Pokemon Diamond Version Trainer AI Patch Plus
Pokemon Pearl Version Trainer AI Patch
Pokemon Pearl Version Trainer AI Patch Plus
Pokemon DP-level Platinum
Pokemon DP-level Platinum Plus

These were originally made to fix a bug found in the original Diamond and Pearl games, but got expanded to making more challenging versions of Diamond, Pearl and Platinum based on ideas I had since a long time ago.
